# ONLINE PAYMENT FRAUD DETECTION

Minor Project for the University Degree Course, using Django and Machine Learning to detect fraudulant transactions in the given data.

## Requirements

-[Django]
-[Pandas]
-[sklearn]

## How to run

-[Copy the code, or pull from git.
Setup the python venv
Install the requirements
Run server using:]

```bash
python manage.py runserver
```
